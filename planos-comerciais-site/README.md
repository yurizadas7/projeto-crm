# Apresentação Comercial EMBER SaaS

Página comercial em HTML para apresentar os produtos EMBER CRM, Barbearia Ember Studio e Pizzaria SaaS com visual profissional, responsivo e pronto para publicação.

## Objetivo atual

- manter a página limpa e orientada para conversão
- destacar o plano Profissional como principal oferta
- reforçar o CTA de proposta comercial no topo e no final
- reduzir blocos repetitivos para melhorar a leitura e a venda

## Estrutura

- `index.html`: vitrine principal dos planos comerciais.
- `crm/index.html`: apresentação individual do EMBER CRM.
- `pizzaria/index.html`: apresentação individual da Pizzaria SaaS.
- `contato/index.html`: perfis comerciais e links de atendimento pelo WhatsApp.
- `css/style.css`: identidade visual preto/dourado e layout responsivo.
- `js/app.js`: animação fluida do fundo.
- `assets/`: logo EMBER e fotos otimizadas dos perfis comerciais.

## Próximos refinamentos recomendados

1. deixar o plano Profissional como destaque visual constante em todas as seções.
2. criar uma comparação rápida entre os planos para facilitar a decisão.
3. manter uma linguagem mais forte e direta de venda em cada bloco de produto.

## Publicação

O site pode ser publicado via GitHub Pages usando a branch `main` e a pasta raiz do repositório.
