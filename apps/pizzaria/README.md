# Organizacao do modulo Pizzaria

## Cliente final

- Entrada: `cliente-login/index.html`
- Cardapio protegido: `cliente/index.html`
- O cliente precisa criar perfil com nome, WhatsApp, bairro, rua e numero antes de acessar o cardapio.

## Restaurante/Admin

- Entrada com codigo e PIN: `acesso/index.html`
- Painel administrativo: `admin/index.html`
- Demo admin: `YULIPE-001 / 123456`

## Observacao

Esta organizacao esta apenas no front-end. Os dados do perfil do cliente ficam no navegador via `localStorage`.
